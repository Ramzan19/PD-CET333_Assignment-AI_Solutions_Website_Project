## Portfolio Items Still Requiring Personal Evidence

These items cannot be completed by code alone and should be added to the final PDF report:

- Student name and ID on the front cover.
- Signed requirements specification.
- Three signed client contact record sheets.
- Project schedule with any real changes made during development.
- Screenshots from testing and evaluation.
- Deployment screenshots or screencast link.
- Critical reflection written in your own words.
- Harvard references for any external sources used in the report.
