# Prototype Demonstration Script

Use this as a checklist while recording the Task 2 video.

## 1. Open the Public Website

Show the homepage and explain that AI-Solutions is a Sunderland-based AI software company prototype.

## 2. Show Services

Open `services.php` and show:

- Virtual Assistant.
- Workflow Automation.
- Data Analytics.
- AI Product Prototyping.
- Past solution examples.

## 3. Submit a Customer Inquiry

Open `contact.php`, fill in the required fields, choose a solution interest, submit the form, and show the success page.

## 4. Schedule a Demo

Open `schedule-demo.php`, fill in the required fields, select an interested solution, submit the form, and show the success page.

## 5. Join an Event

Open `events.php`, show the upcoming events, use a Join Event link, fill in the registration form, and show the success page.

## 6. Use the AI Assistant

Open the assistant, ask about services or pricing, then request human handover. Submit the handover form with name, email, phone, company, country, topic, and summary.

## 7. Submit Feedback

Open the Contact page feedback section, submit a rating, and show that feedback appears on the site.

## 8. Admin Login

Open `admin/login.php` and sign in with the local testing credentials from `README.txt`.

## 9. SMTP Email Setup

Open SMTP in the admin navigation. Show that real SMTP can be enabled and tested with a provider such as Gmail, Outlook, or Yahoo using an app password.

## 10. Admin Dashboard and Analysis

Show:

- Customer demand records.
- Demo bookings.
- Event registrations.
- Assistant interest.
- Prototype interest.
- Demand by country.
- Interest mix.
- Feedback rating summary.
- Visitor tracking.

## 11. Admin Record Management

Show one record list, then view, edit, export, email, and delete a test record if appropriate.

## 12. Close the Demo

State that customers do not create accounts and that only the admin area is password-protected.
